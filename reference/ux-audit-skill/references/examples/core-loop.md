# Core Loop: Sign In → Vote → Leaderboard

**App URL:** http://localhost:3000
**App Name:** SavingForBonding
**App Persona:** A football prediction game for a group of friends. Users sign in, browse upcoming matches, cast votes on outcomes, and track beer debts on a leaderboard.
**Auth:** email: ux-audit@dev.local / password: devpassword123
**Session:** fresh

## Scenario

You are a friend who has just been invited to this app for the first time. You have never used it before. Walk through the critical path:

1. Arrive at the sign-in page cold. Note your first impression — is it obvious what this app is?
2. Attempt to sign in with a wrong password. Observe the error message.
3. Sign in successfully with the correct credentials.
4. Land on the home screen. Without any guidance, try to understand what to do next.
5. Find an upcoming match that has not been voted on and open it.
6. Cast a vote for one of the teams. Observe the confirmation state.
7. Navigate to the leaderboard. Find your own name in the rankings.
8. Navigate back to home and note whether it is easy to return.
